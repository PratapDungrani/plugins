== Description ==

Woo Mailjet plugin allows the business owner to collect user email and store it in the form of contact of mailjet. Buyer is provided with checkbox option on checkout page to subscribe to new later. If user has checked the check box, it will create contact into mailjet and will send email notice to admin.


== Settings ==
From Woocommerce email tab, admin can manage mailjet settings
1) Add api and secret key of mailjet
2) Set email options
